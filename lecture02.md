# 第２回課題

## 第２回講座メモ
### Git
- バージョン管理システム
- リモートリポジトリ（GitHub）とローカルリポジトリ（Git）の分散型

### コマンド git~
- clone リモートのデータをローカルへ複製（初回）
- pull ２回目以降の複製
- add 変更を加える
- commit ローカル内で変更を反映
- push ローカルの変更データをリモートへ反映

※ ブランチ（分岐）及びマージ（統合）については復習要！
